# Introduction

This is first and foremost an educational project to help my understanding of the Python programming language by developing a turn-based RPG. As of starting this project, I believe I have a good understanding of Python, would consider myself an early intermediate-level programmer, but lack the experience of developing a full project from start to finish using various third party libraries, modules, and object oriented programming. I intend to avoid the use of AI generated code in this project, and will instead do my best to rely on my own knowledge, research, and problem solving skills to develop this project. 

# Structure

This project will sort all objects (Not python OOP) into a single folder and into respective .json files. Each object represents a a single entity in the game, such as an item, piece of equipment, or ability.
